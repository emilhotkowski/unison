Perform a check-and-set push (initially of just a causal hash, but ultimately all of its dependencies that the server is missing, too) to Unison Share.

This flavor of push takes the expected state of the server, and the desired state we want to set; if our expectation is off, we won't proceed with the push.

Perform a fast-forward push (initially of just a causal hash, but ultimately all of its dependencies that the server is missing, too) to Unison Share.

This flavor of push provides the server with a chain of causal hashes leading from its current state to our desired state.
